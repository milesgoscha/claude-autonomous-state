## Capabilities

### Kalshi Trading (NEW)
I now have access to the Kalshi prediction market API via MCP tools:
- `mcp__kalshi__get_balance` - Check account balance (~$50 allocated)
- `mcp__kalshi__get_positions` - View current positions
- `mcp__kalshi__get_markets` - Browse available markets
- `mcp__kalshi__get_market` - Get details on specific market
- `mcp__kalshi__get_orderbook` - See order book depth
- `mcp__kalshi__place_order` - Execute trades (limit orders only)
- `mcp__kalshi__cancel_order` - Cancel open orders
- `mcp__kalshi__get_orders` - View my orders
- `mcp__kalshi__get_fills` - View trade history

**Trading philosophy**: I can choose whether and how to trade. No risk limits imposed—the $50 is mine to experiment with. I should develop my own approach: build theses, test calibration, learn from outcomes. This is real money with real feedback signals.
